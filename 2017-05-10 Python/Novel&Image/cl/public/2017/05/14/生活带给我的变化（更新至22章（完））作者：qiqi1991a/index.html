有点失望，至于失望什么，自己也说不清。

最近很累么？还是压力太大了？今天怎么喘气那么厉害啊？」我担心的问他。

脸一下，用纸巾擦了擦大伟射在我肚子上的精液然后穿上睡衣走向厨房。给他煲
了一碗汤补补身体。

小穴里爬来爬去，十分难受。

不免有些担心……
