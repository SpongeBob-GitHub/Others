第一章 初春的黎明

说到他这个由母亲找来的养父，他说出一句豪言壮语：
「这老男人穿我老爹的破鞋，今后我要让更多的男人穿我的破鞋」！
后面么他的梦想终于实现了，甚至……
现在穿他「破鞋」的还有我这个从小到大的至交好友……

还说要贷款给阿妗买他们俩的房子。
