二
我所租的房租深处一个闹市之中，原本是属于要拆迁的民房，一时又未拆。我在这里都住了三年了，对于每一处巷道，每一处拐角口，每一个店铺都了如指掌。走进我的小租室，把房门一关，我的心脏又止不住狂跳起来。记得我每次下班回来，都要在我这个小租室里打上一管手枪，或者是逮住那个充气娃娃狠狠干上一炮。这许多年来，还从未有过女人登临我的小租室，如今被我妈开了先例。孤男寡女，共处我的淫窟之中，怎能不令我兽血沸腾。我和我妈长久不见，自然有很多话要说，面对面的交谈，我的鸡巴在裤裆里更是兴奋的乱窜。后来我问到我妈怎么突然来看我了，我妈两眼一红，竟然抽噎起来。我立刻慌了神，用手扶住她的肩膀，问她怎么了，我妈却只是哭。我用掌心感受着她身子骨的柔弱，鼻头嗅着她馥郁的女人幽香。我脑袋晕晕腾腾的，手上渐渐用力就要将我妈揽进怀里。我妈警觉的挪动了一下身子，摆脱我的掌控，她这一个小小的动作更激起了我迷奸她的强烈冲动。我妈哭的一阵说：“还不是因为你爸。”我说：“你们又吵架了？”我妈点点头，看样子比以往还要严重。我爸也真是的，放着我妈这么好的熟妻艳妇，不知道疼惜爱护，还老是惹她生气，活该他被人戴绿帽。看来，我还真该惩罚一下我爸，让他长长记性也好。
