小芳慢慢的爬起来之后继续盯着那坨狗屎。而老吴则绕到小芳的身后抓起长裙一下就肏到肉穴里面。老吴可不管那么多，一边肏，一边拍着小芳的屁股，而且手越来越重。我从一开始听不到声音，到后来基本上可以清晰听到啪啪啪。小芳不敢再犹豫，看着这坨狗屎就像美食一样。全部吃下去。回头看了一眼老吴，似乎想对老吴说，我已经吃完了，你就放过我吧。

可是老吴怎么会这么轻易放过小芳呢，浅浅一笑之后，继续肏下去，只是停止了拍打屁股。之后看到老吴又一次射到小芳的肉穴里面，然后看了看小芳的屁眼之后手指用力的推进去。然后老吴将自己的鸡巴塞到小芳的屁眼里面一动不动。

我想了很久怎么会不动呢，可是看着老吴的样子，过了一会我就反应过来了，撒尿！

老吴直接在小芳的屁眼里面撒尿，在满满一肚子硬币，沙子，还有小芳内裤的屁眼里面撒尿。一般射完之后要撒尿是需要点时间的。所以整整站了一分多钟。

然后两个人就继续往宿舍的方向走。我很聪明的坐在小芳宿舍的门口，假装自己睡着。我听到一个人的脚步声，很显然，小芳是趴着上来的。我偷偷的咪一条缝。看到两个突兀的表情，也许是他们都没有想过我会在这里。小芳看到我第一眼就站了起来。这点让我非常的感动。而老吴看到小芳站起来之后就狠狠的两巴掌就甩过去。我假装动了一下，这一次老吴也慌了。他立马解开脖子的腰带直接绑到小芳的腰上，我看到小芳的肚子微微的突出来。我假装醒过来。站起来第一句就问，「回来了？」

小芳和老吴也和我寒暄一下，说天气有点晒，都晒伤了，晒肿了。还吃东西吃太多，老吴还故意指了指小芳的肚子说，你看都吃的涨涨的。我也知趣的没有说些什么，即使我知道这两天小芳基本吃的全是精液，肚子里最多的是硬币。最后和老吴一起坐车走了。毕竟我怕老吴继续虐待我心爱的小芳。不过其实一切已经无能为力了不是吗？

一切好像回到正常状态，平时也会和小芳出去约会什么的。直到两个月后的一个凌晨两点，我接到小芳的一个电话，电话里面小芳说着想我，还说着一些工作上困难的事情。可是我依稀听到啪啪啪的声音，还有小芳说话时候挺模糊的，我想，估计老吴正在肏着小芳。不过其实我错了。

当时，小芳赤裸着身体，双手绑在身后。老吴拿着手机放在小芳的耳边，而自己的鸡巴就直接抽插着小芳的嘴巴，小芳只能靠着偶尔说点什么话。而小芳下面躺着的是房东，一个五十岁的老男人，啤酒肚直接顶着小芳的肚子。双手抓住小芳的奶子，指甲狠狠的抓到肉里面，屁眼给楼上一个租客插着，这个租客同时也是小芳的同事，这个同事双手拍打着小芳的屁眼和后背，想拍哪里就是哪里。

这些，我一切都不知道，而我更不知道的是，小芳当天下午才去做完人流手术。唯一知道的是，小芳的世界，已经是属于老吴的了。

其实也不是属于老吴的，因为到了几个月后，小芳，每天都被肏。 上班被自己的同事，老板，甚至有时候客户也会肏。 下班回到宿舍就被房东，被整栋楼的男住户肏。 甚至平时出去玩，也许是老吴带的，也许是同事带的，也会被一些不认识的人带出去，相同的结果就是被更多不认识的人肏。 最多的时候，一年里面做了四次的人流。
