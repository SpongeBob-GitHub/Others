后来一年里我俩经常偷偷出去开房，我让我那个牛粪朋友戴了巨大的绿帽子。当然报应就是我女友也给我戴了绿帽子。很疯狂的绿帽子，偷了一年多情，就像小英下面被我干的都有点发深。
