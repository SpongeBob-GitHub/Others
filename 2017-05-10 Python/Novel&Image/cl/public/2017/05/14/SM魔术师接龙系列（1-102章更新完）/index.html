电压。

每3 秒电椅上的电极将会刺激刘芬一次。刘芬身上会接上9 个电极，分别在
刘芬的阴道，肛门，舌头，双乳，四肢上。电椅是用来执行死刑的，由着名的监

雷管的引线会在电椅的启动同时点燃。刘芬不仅要抵抗电刑的刺激脱身还要

手脚的拇指和食指之间虎口的部分。最后2 个金属制成的阴茎棒和肛门棒插入刘

她们又成功了！
