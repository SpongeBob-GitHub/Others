更狗血的还在后面，过了一段时间，婷的男友单独联系到我前女友，不知道是威胁还是怎么，反正他把前女友约出来操了几次。再后来，这家伙竟然跟婷讲了这个事情，而婷居然没有生气，于是乎在一次他又约我前女友的时候，前女友一进门发现婷也在，当时她就楞了，后来婷和她男友各种洗脑，当晚就来了个双飞。而这一切，仅仅在我前女友破瓜后一年不到。我算了一算，妈的，我竟然顶了两年多的绿帽。
后来这两年时间，前女友被婷调教的简直成了一个荡妇，婷给我说，后来婷的每一个男友都玩过双飞，甚至有时候，婷的某个前男友还叫过几个朋友来一起操她们俩。
更夸张的是有一段时间，为了钱，婷拖上她，两个人给一个富二代当性奴，什么遛狗，调教，sm玩了个遍。就到我和婷聊天那时，前女友已经会自己出去找男人玩了。

当然，聊完这些，我连想上婷的心情都没了，接下来短短时间内，找了个借口，和前女友说了拜拜。事情就到此告一段落，在之后，我都没有和婷再聊过。
