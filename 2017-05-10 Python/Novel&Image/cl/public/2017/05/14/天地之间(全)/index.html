在赵志的办公室里，白秋简单介绍了自己的情况，递上自己的文凭，赵志看了还是比较满意，又简单问了点制药方面的问题，白秋炒了炒陈饭，回答得还不算太离谱，于是赵志当场拍板让白秋先锻炼锻炼，在配料室担任原料保管兼配料工，而吴文确实没两刷子，只好当装卸工凑数，两人终于在这里初步安顿下来了。
