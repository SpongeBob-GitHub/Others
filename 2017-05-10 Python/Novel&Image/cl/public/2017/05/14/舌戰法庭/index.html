這是一個案情相當簡單的強姦案。大約四個月前，陳小姐在半夜回到自己的公寓裡遭到一個預先埋伏的歹徒的攻擊和強姦。在整個強姦過程中陳小姐被蒙上了雙眼，自始至終都未能見到強暴犯的長相。但她卻清晰地記住了罪犯的聲音。
