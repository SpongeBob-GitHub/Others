# 自动更新的小说网站
同时追几本小说，每次打开一个个看是否更新太麻烦写个Python脚本，crond 2分钟运行一次，循环检查是否更新，一旦更新就把推送到github上，然后触发travis自动构建Hexo上，把网页自带推送回github pages上，绑定自己的域名。以后就可以打开这个 自己的网站 查看最新的小说内容了

## 增加标签
为同样喜欢看书的好基友添加一个标签 通过标签区分我们两个看的书

## 增加分类
每本小说即为一个分类
